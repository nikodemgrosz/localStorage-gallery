# Real Madryt - Ulubieni Piłkarze ⚽⭐

Prosty projekt frontendowy z dynamicznym systemem polubień. Pozwala wybrać ulubionych zawodników i zapisuje ich na osobnej podstronie.

## Co to robi?
* **Lista zawodników** – generuje kafelki z piłkarzami (zdjęcie, nazwisko, ikonka serca) prosto z tablicy obiektów w JavaScript.
* **System polubień (LocalStorage)** – kliknięcie serduszka zmienia status zawodnika. Aplikacja zapisuje ten stan w pamięci przeglądarki (`localStorage`), więc po odświeżeniu strony nic się nie kasuje.
* **Zakładka "Ulubione"** – osobna podstrona, która czyta dane z pamięci i wyświetla tylko tych piłkarzy, których polubiłeś.

## Jak uruchomić?
Projekt to czysty frontend. Nie potrzebujesz XAMPPa! 
Po prostu otwórz plik `realmadryt.html` bezpośrednio w dowolnej przeglądarce internetowej.

## Zrobione w:
